Gemini API Key

Used for Google's Gemini AI model

How to get it:
  1. Visit Google Cloud Console
  2. Search for "create project" and click the resulting link
  3. Name your project (e.g., "chrome-extension-vcf-extractor")
  4. Go to APIs and Services panel and click "Create Credentials"
  5. Set Application Restrictions to None
  6. Restrict Key to "Generative Language API" and Save
  7. Find the credential in the list and click Show Key
  8. Copy and paste the key here

OpenRouter API Key

Use OpenRouter's free Deepseek model for contact extraction

How to get it:
  1. Visit OpenRouter Keys Page
  2. Click "Sign in with Google" to create an account
  3. After signing in, click "+ Create Key"
  4. Enter a name for your key (e.g., "Contact Extractor")
  5. Copy the generated API key and paste it here
